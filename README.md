# XiaoTiEquipments（RimWorld1.6MOD）

> **⚠ 未发布版本 | 早期开发阶段 | 请勿分发**

一个以《Helldivers 2》为主题的 RimWorld 1.6 内容模组，添加了大量 HD2 风格的武器、装备、建筑、机制与音乐。

---

## 开发状态

本模组处于 **早期开发阶段**，仅以满足功能需求为目标，**不考虑平衡性与游戏性**。API、Def 命名、功能范围在后续版本中均可能发生破坏性变更。

### 资产版权声明

- **贴图**：大量使用 RimWorld 官方素材；小部分来自其他创意工坊模组（**暂未取得授权**）；少数贴图为 AI 生成。
- **音乐**：来自《Helldivers 2》原声及相关素材。

> 由于上述原因，**本模组暂时不会在 Steam 创意工坊发布**。当前 GitHub 仓库仅供个人开发与版本管理使用。

## 自定义机制（C# 组件）

### 建筑自修复（`CompBuildingSelfRepair`）
建筑按 tick 自动恢复 HP，支持固定数值或百分比两种模式。应用于本模组几乎全部建筑，如纳米修复墙体、炮塔、电池等。

### 半径伤害（`CompProperties_RadiusDamage` / `Projectile_RadiusDamage`）
子弹命中/爆炸时对指定半径内所有目标造成一次不可回避的额外伤害，该范围内所有单位额外结算一次伤害且不会被建筑遮挡。应用于 400mm 炮弹和 40mm 机炮弹。

### 装备赋予 Hediff（`CompApparelAddHediff`）
装备/卸下装备时自动添加/移除指定 Hediff。用于护盾背包赋予"超越生死"等效果。

### 远程护盾（`CompRangedShieldBelt`）
能量护盾系统，拦截远程与爆炸伤害，可破裂后恢复。

### 手动控制炮塔（`TurretCanForcedTarget`）
允许玩家手动指定炮塔强制攻击目标，暂时仅用于本模组炮塔。

### 死亡拒绝（`Patch_Protection_Pawn_Kill_Refused`）
拥有"超越生死"（Cross_The_Bleach）Hediff 的角色在被击杀时阻止死亡，清除所有伤痕与负面状态，添加"战斗兴奋"BUFF，高额减伤同时不会被代码杀（甚至无法离开地图或者进入空投舱，不过持续时间很短以秒为计）。

### 煞气固锁（`HediffComp_MalevolentCap`）
限制螈僵模组（Axolotl）煞气不超过容量上限的指定百分比，螈僵装备CQC-5战斗短柄斧后生效。
